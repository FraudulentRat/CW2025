HOW TO START THE GAME

In VSCode, you just click the run button at the top right corner to run the game.

---------------------------------------------------------------------
ALL FUNCTIONALITIES

1. Movement: 
    a. For WASD:
        i. W key - Rotate the blocks
        ii. A key - Move blocks left
        iii. D key - Move blocks right
        iv. S key - Move blocks down

    b. For Up Down Left Right arrow keys:
        i. Up key - Rotate the blocks
        ii. Left key - Move blocks left
        iii. Right key - Move blocks right
        iv. Down key - Move blocks down

2. Start game:
Clicking the "Start Game" button in the main menu starts and loads the game

3. Sound:
    a. moving blocks will result in a sfx to be played
    b. rotating blocks will also result in the same sfx to be played
    c. completing a line of tile will play another sfx
    d. When the game ends, BGM will be stopped to play the game over sfx

4. BGM:
There are 2 Background music, one playing during the main menu and another during the game itself. Swapping between main menu to game will cause BGM to switch, in-game BGM will stop once the game over screen is displayed.

5. Points:
    a. Only scoring points when blocks form a horizontal line, of which will lower every tile above the line down one level
    b. ScoreBoard will show the total accumulated points
    c. This is the only way to score points in this game
